PII-Anonymizer Web-CLI

Detect, Anonymize, and Protect Personally Identifiable Information

How to Use:

1. Enter text directly OR upload a file (image, PDF, TXT)
2. For files: Add a context prompt to specify which PII to extract
3. Select an anonymization mode
4. Click "Anonymize & Process with LLM" to protect PII and get AI response
